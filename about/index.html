Every so often my design tastes change, and therefore this website does. My design/coding skills also get better over time so I can do more cool things with this website and other peoples.

I believe that water, privacy, democracy, freedom of speech (to an extent), open and free software, a fast and reliable internet connection, a climate that doesn't change, affordable healthcare, education, a living wage and more are human rights.

about this site

Every so often my design tastes change, so this website changes with it. You're on the latest incarnation, which my versioning system calls "experimental v3.0". I'm still coding the latest incarnation of the site. This redesign is special, because it's built from the ground up with a new layout system (cards) which makes this easy for me to add new parts to the website.
